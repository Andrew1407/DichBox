## Setup

Firstly, install dependencies:
> npm i

Create **.env** file ([example](https://github.com/Andrew1407/DichBox/blob/main/client/.env.example)).

Run tests:
> npm test

Run client in development mode:
> npm start

Build *./src* with:
> npm run build

The artefact will be built in *server/dist* directory.
